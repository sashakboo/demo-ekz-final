Приложение использует в качестве БД облачную субд Postgres. Подключение уже настроено в приложении. 
При необходимости параметры подключения к БД можно изменить в файле `.\backend\.env`.

При запуске автоматически инициализируются таблицы. При необходимости это можно сделать вручную выполнив скрипт `./init-db.sql`.

Для запуска приложения выполните в текущей директории следующие команды.

Установка npm-зависимостей:

```bash
npm run install-all
```

Параллельно запустить клиент и сервер
```bash
npm run dev
```

Сайт должен открываться по адресу `http://localhost:3000/login`

Для входа под аминистратором используйте логин `Admin26` и пароль `Demo20`.
